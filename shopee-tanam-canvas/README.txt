Kebun Hadiah Canvas

Cara menjalankan:
1. Buka file index.html langsung di browser modern.
2. Atau jalankan server lokal dari folder ini, lalu buka alamat lokalnya.

Kontrol:
- Pilih Benih/Air/Pupuk di bagian atas.
- Tap lahan untuk menanam, menyiram, memupuk, atau memanen.
- Buka Toko untuk membeli item.
- Buka Misi dan Bonus untuk klaim hadiah.

Game menyimpan progres otomatis lewat localStorage browser.

Aset hasil generate:
- assets/splash-garden.png: splash/background layar awal.
- assets/gameplay-bg.png: background utama gameplay ukuran canvas 390x844.
- assets/gameplay-bg-morning.png: background gameplay pagi.
- assets/gameplay-bg-day.png: background gameplay siang.
- assets/gameplay-bg-evening.png: background gameplay sore.
- assets/gameplay-bg-night.png: background gameplay malam.
- assets/gameplay-time-preview.jpg: preview 4 scene waktu.
- assets/item-atlas.png: atlas 12 ikon transparan.
- assets/icons/: ikon PNG terpotong per item.
- assets/crop-stages.png: atlas transparan fase tanaman.
- assets/crop-types-atlas.png: atlas transparan 6 jenis tanaman x 5 fase growth.
- assets/crop-types-preview.jpg: preview semua sprite tanaman per jenis benih.
- assets/crop-types-gameplay-preview.jpg: preview enam jenis tanaman saat ditempatkan di lahan gameplay.
- assets/environment-atlas.png: atlas transparan tanah, maskot, efek, dekorasi.
- assets/sprites/: sprite gameplay terpotong per objek, termasuk benih, sprout, tanaman besar, berbuah, soil plot, splash, sparkle, dan maskot.
- assets/asset-preview.jpg: preview cepat semua ikon.
- assets/gameplay-asset-preview.jpg: preview cepat semua sprite gameplay.
- assets/gameplay-composite-preview.jpg: preview komposisi gameplay dengan sprite di atas background.
